You can download the link to the original RaiderPlanner project below.

  • Download link from GitHub
  • Extract the file into your desired location.
  • Start Eclipse and import your file.
  • Run RaiderPlanner.git to start the program.

There is also a development tool chain that may be useful to new users.

You may use any software that you wish, this list is just suggestions. You will at least need a JDK, IDE and access to GIT Version control.

  • For linux a JDK like OpenJDK should already be installed on your system. For Windows and Mac you will need to download a Java installer from oracle.Here is a link!
  • As far as choosing an IDE goes we recommend Eclipse with the following plugins, Gradle BuildShip, Data Tools Platform SDK/Data Tools Platform Extender SDK, CheckStyle,FindBugs
  • As stated before the RaiderPlanner project is based on GIT version control.
